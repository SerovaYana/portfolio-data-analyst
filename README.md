# portfolio-data-analyst
Портфолио проектов для стажировки на позицию аналитика данных. Здесь представлены учебные и практические кейсы по исследованию данных, очистке и визуализации. Стек: Python (Pandas, NumPy, Matplotlib, Seaborn) для анализа, SQL (PostgreSQL) для запросов, YandexDataLens для дашбордов.
